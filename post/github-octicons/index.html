⚓ Github Octicons

These are absolutely gorgeous. Congratulations to Github on coming up with a rather interesting and well implemented solution to their problem. With the state of displays being in flux as it is today, it makes sense to offload the rendering to the OS as much as possible, effectively future proofing another aspect of their website.

While this might not be a solution for most of us, it seems like the best way to distribute icons (and glyphs in general) that scale well across all resolutions and sizes. Maybe we should start thinking differently about what constitutes a "font", and expand on how we use them beyond the alphanumeric symbols.

UPDATE: Looks like I've been living under a rock. Turns out Icon Fonts are a real thing. Suddenly my epiphany doesn't seem so epiphanic.
